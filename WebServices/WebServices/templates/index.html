Overview

The SATNet network is a software development project hosted at GitHub. Its objective is to develop the software for the nodes of a communications network that shares Ground Stations among CubeSat operators.

A simplified description for the network topology is provided in the figure on the right. The SATNet network is intended to be composed of several independent subnetworks that can be interconnected, either for sharing resources or for redundancy.

The server nodes are the core for each subnetwork, enabling the sharing of the resources of its client nodes across other subnetworks. Ground Station owners will define their availablity and sharing rules, so that restrictions may apply when connecting those facilities to Spacecraft Operations Centres. Subnetworks may be private if so defined, restricting the share of resources only in between its members.

The complete description of the system can be found in the wiki site enabled at Wiki.
